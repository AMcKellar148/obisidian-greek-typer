# Obsidian Greek Typer Plugin

A plugin for Obsidian that allows you to type Greek letters and polytonic Greek diacritics easily using simple Latin transliteration sequences. Supports live typing conversion, selection conversion, and a handy key guide.

---

## Features

- **Greek letters:** Type `a` → `α`, `b` → `β`, etc.
- **Polytonic diacritics:** Add marks like acute `/`, grave `\`, circumflex `=`, smooth breathing `)`, rough breathing `(`, iota subscript `|`, and diaeresis `+`.
- **Live typing:** Convert words automatically as you type.
- **Selection conversion:** Convert selected text to Greek or polytonic Greek.
- **Typing guide:** Right pane guide shows all key mappings and examples.
- **Full polytonic support:** Combine marks to create accented letters (e.g., `α)/` → `ἄ`).

# Usage

- Convert selection to Greek: Select text and run - Convert to Greek command.

- Convert selection to Greek with diacritics: Select text and run Convert to Greek (with Diacritics) command.

- Live typing: Toggle via ribbon icon to automatically convert words as you type.

- Typing guide: Open the right pane guide via command Open Greek Typer Guide to see key mappings and examples.
